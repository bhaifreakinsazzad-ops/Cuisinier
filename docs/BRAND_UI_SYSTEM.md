# Cuisinier Brand UI System

## Visual Direction

- Midnight AI kitchen
- Premium dark surfaces
- Neon orange action color
- Clean mobile-first conversion flow
- Motion focused on splash, home, craving, and discovery

## Core Colors

- Background: `#080808`
- Surface: `#111111`
- Primary: `#ff7a00`
- Text: `#f5f5f5`
- Muted text: `rgba(255,255,255,0.6)`
- Success: `#22c55e`
- Error: `#ef4444`

## Component Rules

- Use glass panels for cards and modal surfaces
- Keep checkout readable and low-friction
- Use orange glow only on high-priority CTAs
- Keep support, admin, and tracking layouts calmer than the home and craving screens

## Motion Rules

- Heavy motion: splash, hero, craving scan, menu discovery
- Light motion: cart, checkout, admin, tracking
- Avoid animation that slows checkout or obscures tap targets

## Responsive Rules

- Mobile first
- Bottom navigation on small screens
- Comfortable touch targets
- No tiny text on checkout or admin controls
- Desktop should expand into wider card grids without changing the product model
