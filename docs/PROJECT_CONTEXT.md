# Cuisinier Project Context

## Brand

- Name: Cuisinier
- Tagline: AI-Powered Cloud Kitchen
- Brand line: Late-Night in Dhaka • Daytime in Dhanmondi

## Business Model

Cuisinier owns the digital brand, menu presentation, customer ordering flow, tracking, support, and admin operations. Cooking and delivery are handled by a third-party partner.

## Product Priorities

1. Fast guest ordering
2. Menu discovery and craving selection
3. Manual payment support
4. Order timeline tracking
5. Admin operations and forwarding
6. Installable mobile-first PWA UX

## Service Rules

- Night kitchen: 11 PM to 4 AM
- Daytime service: Dhanmondi and nearby areas
- Delivery fee: `৳100` by default
- Payments: COD, bKash manual, Nagad manual
- bKash and Nagad require sender number and transaction ID
- No fake GPS tracking
- No forced customer login

## Customer Flow

Home -> Craving selector or menu -> Customize item -> Cart -> Checkout -> Order created -> Tracking timeline -> WhatsApp support if needed

## Admin Flow

Admin login -> Order overview -> Order detail -> Status updates -> WhatsApp forward -> Menu CRUD -> Settings
